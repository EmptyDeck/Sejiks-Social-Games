Universal GitHub to rawcdn.githack.com Converter

Supported URL Formats:

1. Standard GitHub file URL:
https://github.com/user/repo/blob/main/path/file.html
2. Raw GitHub URL (with refs/heads):
https://raw.githubusercontent.com/user/repo/refs/heads/main/path/file.html
3. Raw GitHub URL (direct):
https://raw.githubusercontent.com/user/repo/main/path/file.html
Output for all formats:
https://rawcdn.githack.com/user/repo/main/path/file.html
